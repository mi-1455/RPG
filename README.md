# RPG_裏世界を体験しよう！

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## 世界観
* 操作キャラ
* 　特攻隊所属
* 　強さは中の上
* 　小隊メンバー4人うち1名死亡
* 　攻撃力：2~3桁
* 　HP：3桁
* 操作キャラA
* 　剣使い
* 　能力：身体強化
* 操作キャラB
* 　銃使い
* 　能力：弱点看破
* 操作キャラC
* 　能力使い
* 　能力：火炎操作

* 敵NPC
* 　1回の遭遇上限：3体
* 　進むごとに強くなる
* 　一定範囲に入ると感知して追ってくる
* 　種別ごとに行動パターンが異なる（視力で感知、聴覚で感知など）
* 　攻撃力：2~4桁
* 　HP：3~4桁

* MP
* 　障害物の有無
* 　浸食率の増加
* 　MAX：5ステージ（3ステージくらいで死ぬような難易度の予定）

## ゲームの概要
* 一本道の右スクロールの戦闘メインのRPG
* ステージが5まで存在し、ステージごとに1回敵と遭遇する
* 敵NPCと操作キャラが接触した場合、戦闘画面へ移行し、戦闘開始
* 逃げることに成功すると、前のステージに戻された上で、前ステージの敵との強制戦闘（ランダムで個体数が変化）
* 最初のステージで逃げることに成功すれば、一定確率で味方本体と接触でき、生存ルートに入る（隠し√）
* 逃げた上でまた進むと敵の個体数が増加している
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 矢印キーを操作してMP内を移動、敵NPCと接触した段階で戦闘開始
* 戦闘：戦闘画面に移行し、矢印キーでコマンド選択をし、行動する
* 戦闘終了：操作PC、もしくは敵NPCのHPが0以下になった段階で戦闘終了
* 戦闘画面：戦闘画面には敵キャラの画像と行動順の表記、こうげき、ぶんせき、にげる、といったコマンドがある

## ゲームの実装
### 共通基本機能
* 背景画像の描写
* 主人公PC、敵NPCの描画

### 分担追加機能
* END処理
* 侵食率
* しんがり
* イベント
* モード選択
* 実績解放
